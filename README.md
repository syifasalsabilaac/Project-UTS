# Website jual beli sederhana
Website ini dirancang untuk memperjualbelikan berbagai merchandise dari sebuah grup K-pop Hearts2Hearts.

## Feature website
- Halaman Home
- Halaman Product
- Halaman Contact

## Tech
Aplikasi ini dibangun menggunakan:
- VSCODE
- HTML
- CSS
- JAVASCRIPT
- BOOTSTRAP"# Project-UTS"
